Triangular mesh viewer class using libigl (C++)
============================================

Simple implementation based on the tutorials.
--------------------------------------------------------------------------
To execute...
```sh
$ mkdir build
$ cd build
$ cmake ..
$ make -j2
$ cd VisualMesh-bin
$ ./VisualMesh ../../objects/horse_quad.obj
```
