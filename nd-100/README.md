ND-100 GPS Dongle class (C++)
============================================

Implementation to get data from USB Dongle.
--------------------------------------------------------------------------
To execute...
```sh
$ mkdir build
$ cd build
$ cmake ..
$ make -j2
$ cd ND100-bin
$ ./ND100.bin
```
